Solution: To find the total income we would want to integrate \((50+5t)\) as \(t\) goes from 0 to 10. To find the current value we must discount the income based in when we receive it. Thus we want to integrate \((50+5t) e^{-0.06t}\) as \(t\) goes from 0 to 10. We set up the integral \(\int_0^10 (50+5t) e^{-0.06t} dt\). Since we do not know how to find the anti-derivative for this function, we find the area either with Excel and Riemann sums, or with Wolfram Alpha.
