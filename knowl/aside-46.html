The point \((a,b)\) is a critical point for the multivariable function \(f(x,y)\), if both partial derivatives are 0 at the same time.

In other words

\begin{equation*}\frac{\partial }{\partial x} f(x,y)|_{x=a,y=b}=0\end{equation*}

and

\begin{equation*}\frac{\partial }{\partial y} f(x,y)|_{x=a,y=b}=0\end{equation*}
