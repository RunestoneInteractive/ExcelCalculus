Math books (and Excel) like using a base of \(e\) because it makes the mathematics easier when we do calculus, so the equation is written as \(f(t)=p*e^{rt}\) or \(f(t)=p*exp(r t)\text{.}\) However, in real world applications we tend to use \(A=pR^t\) and make the equation \(f(t)=p*R^t\text{.}\) (The reader is warned that \(R=e^r\) and both \(R\) and \(r\) are referred to as the rate. You will have to use the context to tell them apart.)
