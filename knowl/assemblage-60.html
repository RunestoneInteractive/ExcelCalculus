Definition: A Probability Density Function is a function that spreads the area 1 over the entire real line, with the obvious understanding that no value can have a negative probability.

In calculus terms, a Probability Density Function is a function \(f(x)\) defined for \(-\infty\lt x \lt \infty \) such that \(f(x)\ge 0\) and \(\int_{-\infty}^{\infty}f(x)dx=1\text{.}\)
