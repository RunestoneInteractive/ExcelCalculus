The more challenging case arises when there is no answer, but we get close. We can ask Goal Seek to find an x with \(1/x^4 = 0\text{.}\) Clearly this problem has no answer. However, if we start with a guess of x = 1, we get an answer of x = 6.14798. That is because \(1/6.14798^4\) is within our tolerance of 0. In both of these cases we see that when we use Goal Seek we should also look at the graph of the function in question to make sure we are asking a reasonable question.
