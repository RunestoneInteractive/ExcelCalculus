Example 4.2.6. Simple derivative of a product.

Let \(f(x)=x\) and \(g(x)=x^2\text{.}\) Find the derivative of \(f(x)*g(x)\text{.}\)

Solution: Note that \(f(x)*g(x)=x^3\text{.}\) Using our rule for monomials \((f(x)*g(x))'=(x^3 )'=3x^2\text{.}\) Using the same rule we see \(f'(x)=1\text{,}\) and \(g'(x)=2x\text{.}\) We can now evaluate using the product rule:

\begin{equation*} \begin{aligned} (f(x)*g(x))' \amp = f'(x)*g(x)+f(x)*g'(x) \\ \amp = (1)*(x^2 )+(x)*(2x)= 3x^2. \end{aligned} \end{equation*}
