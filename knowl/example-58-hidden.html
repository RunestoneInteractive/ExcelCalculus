Let \(f(x)=x^2\) and \(g(x)=x\text{.}\) Find the derivative of \(f(x)/g(x)\text{.}\)

Solution: Note that \(f(x)/g(x)=x\text{.}\) Using our rule for monomials \((f(x)*g(x))'=(x )'=1\text{.}\) Using the same rule we see \(f'(x)=2x\text{,}\) and \(g'(x)=1\text{.}\) We can now evaluate using the quotient rule:

\begin{equation*} \begin{aligned} (f(x)/g(x))' \amp = \frac{f'(x)*g(x)-f(x)*g'(x)}{(g(x))^2} \\ \amp = \frac{(2x)*(x )-(x^2)*(1)}{x^2}=\frac{x^2}{x^2}=1. \end{aligned} \end{equation*}
