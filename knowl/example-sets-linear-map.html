Example5.2.13

Given a \(2 \times 2\) matrix \begin{equation*}A = \begin{pmatrix} a & b \\ c & d \end{pmatrix},\end{equation*} we can define a map \(T_A : {\mathbb R}^2 \rightarrow {\mathbb R}^2\) by \begin{equation*}T_A (x,y) = (ax + by, cx +dy)\end{equation*} for \((x,y)\) in \({\mathbb R}^2\). This is actually matrix multiplication; that is, \begin{equation*}\begin{pmatrix} a & b \\ c & d \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix} = \begin{pmatrix} ax + by \\ cx +dy \end{pmatrix}.\end{equation*} Maps from \({\mathbb R}^n\) to \({\mathbb R}^m\) given by matrices are called linear maps or linear transformations.
