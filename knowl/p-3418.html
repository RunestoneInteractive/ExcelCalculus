Thus, straight-line depreciation gives a value function, which is a straight line. We now plug in known values to find the constants. We plug in for \(time = 0\) to see \(C = 100,000\). We plug in for \(time = 5\) to see \(k = 20,000\). This gives us the same equation using calculus as we obtained using algebra. The book value of the equipment at 3.5 years is $30,000 with this method of depreciation.
